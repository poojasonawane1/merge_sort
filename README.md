## 📚 Algorithms

### Merge Sort

Merge Sort is a comparison-based sorting algorithm that follows the **Divide and Conquer** approach. It recursively divides the array into smaller subarrays, sorts them, and then merges the sorted subarrays to produce the final sorted array.

- **Time Complexity:** `O(n log n)`
- **Space Complexity:** `O(n)`
- **Stable:** ✅ Yes
- **In-Place:** ❌ No
